This directory contains plugins for Radar Tools (RAT), 
These plugins are developed either by RAT developers, or by third parties.
The license is in general not equivalent to the license of Radar Tools (RAT). 
Please consult the sources or additional files for every individual plugin 
for the license agreements.

In order to get these plugins to work with RAT, just copy them in one of the
two directories:
- $RAT_SOURCE_PATH/plugins
- $RAT_INSTALL_PATH/plugins

Under Unix, $RAT_INSTALL_PATH is usually equal to ~/.rat
while $RAT_SOURCE_PATH is the directory where you have copied 
the obtained RAT sources plus the rat directory. 
For example: /usr/loca/rat  or ~/work/rat
